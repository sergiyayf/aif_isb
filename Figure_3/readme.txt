To create a combined barplot for for Pfail, including experimental and simulation data:

- run barplots_Pfail_experiments.py
- run random_walker_sims.py (in ./random_walker/)
- copy sim_stats.pickle to folder of barplot_Pfail_combined.py (exp_stats.pickle should already be there)
- run barplot_Pfail_combined.py with desired sample names
